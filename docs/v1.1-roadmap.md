## Provider 架构（v1.1 新增）

```
LLMProvider (Protocol)
    ↓
BaseProvider (ABC)
    ├── 提供 review() 指数退避骨架
    ├── 抽象 _call(prompt) → str
    └── DEFAULT_MAX_RETRIES=3, DEFAULT_TIMEOUT_S=60
        ↓
        ├── MockProvider（测试）
        ├── DeepSeekProvider（默认生产）
        ├── ClaudeProvider（备选）
        └── OllamaProvider（本地）
```

## 关键设计原则

- **OCP（开闭原则）**：新增 Provider 仅实现 `_call()`，不修改基类
- **KISS**：BaseProvider 只做"重试+超时"，其他行为由子类定义
- **DRY**：Claude/Ollama/DeepSeek 共享同一重试逻辑
- **错误传播**：`_call()` 失败不吞错，由 BaseProvider 统一决定重试
